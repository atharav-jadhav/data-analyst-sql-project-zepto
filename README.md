# Data Analyst SQL Project - Zepto Inventory Analysis

## Project Overview

This project focuses on analyzing Zepto inventory dataset using SQL to extract meaningful insights.  
It demonstrates solving business problems and applying advanced SQL concepts used in real-world data analysis.

---

## Dataset Information

The dataset contains Zepto product inventory details including:

- SKU ID  
- Category  
- Product Name  
- MRP  
- Discount Percentage  
- Available Quantity  
- Discounted Selling Price  
- Weight in grams  
- Stock status  
- Quantity  

Dataset link: [Zepto Inventory Dataset](https://www.kaggle.com/datasets/palvinder2006/zepto-inventory-dataset/data?select=zepto_v2.csv)

---

## Objectives

- Perform data exploration and cleaning  
- Solve business-related SQL problems  
- Practice advanced SQL queries and functions  

---

## SQL Concepts Used

- Aggregations (COUNT, SUM, AVG)  
- GROUP BY & ORDER BY  
- Subqueries  
- CASE statements  
- Data cleaning with DELETE and UPDATE  
- String and numeric functions  

---

## Key Business Problems Solved

1. Find the top 10 best-value products based on the discount percentage  
2. Find products with high MRP but out of stock  
3. Calculate estimated revenue for each category  
4. Find all products where MRP is greater than ₹500 and discount is less than 10%  
5. Identify the top 5 categories offering the highest average discount percentage  
6. Find the price per gram for products above 100g and sort by best value  
7. Group products into categories based on weight: Low, Medium, Bulk  
8. Find the total inventory weight per category  

---

## Outcome

This project strengthened SQL skills, including data cleaning, aggregation, and advanced query techniques.  
It also demonstrated practical analysis for product inventory, pricing, and category-based insights.
